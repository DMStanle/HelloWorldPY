# HelloWorld

This is a standard HelloWorld program. It was made to demonstrate Python, VSCode and GitHub integrated together.

## Instructions for Build and Use

[Software Demo](Put_Your_Video_Link_Here)

Steps to build and/or run the software:

1. Install Python 3.14.2 on your computer.
2. Open the project folder in VSCode.
3. Open the file "helloworld.py".
4. Type "python helloworld.py" in the terminal and hit enter or press the Run button.

Instructions for using the software:

1. Run the program.
2. Observe the output printed in the terminal.
